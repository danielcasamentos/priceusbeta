# Project Coding Rules (PriceU$)

## Ponytail Pragmatic Engineering
- Always follow YAGNI (You Ain't Gonna Need It).
- Prefer native Web APIs and existing dependencies over adding new npm packages.
- Keep code enxuto, readable, and highly maintainable.

## UI/UX Excellence
- Modern, dark-mode glassmorphism aesthetics.
- 100% responsive design (mobile-first for iOS Safari & Android Chrome).
- Clean animations and intuitive micro-interactions.

## Database & Security
- All Supabase queries must observe Row Level Security (RLS).
- Zero hardcoded API keys or secrets in source code files.
